MODULAR PROGRAMMABLE DRONES

We need endurance

Designer

XuYunxiu, LuoShan, ZhaoMi

Date

2018.9

Category

Control

We developed 3 different types of drone in different weight which is realated to the endurance of the drone. We set the flight control board in the form of superimposing modules, and made different control boards with other sensors with extended functions, to change various functions of the aircraft.

We plugged in scratch to program drones.Scratch is a graphical, easy-to-learn programming language that can be used to control drone flights by running it instead of a remote control.
